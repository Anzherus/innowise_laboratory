# Lecture 2 - Mini Profile Generator 
 
A Python program that collects user information and generates a personal profile summary. 
 
## Features 
- Collects user name and birth year 
- Calculates age based on 2025 
- Determines life stage (Child/Teenager/Adult) 
- Collects multiple hobbies 
- Displays formatted profile summary 
 
## Usage 
```bash 
python mini_profile.py 
``` 
 
## Requirements 
- Python 3.6+
